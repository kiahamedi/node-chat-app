# Node Chat App
> Chat room web app built on node.js, powered by express server with socket.io to enable real-time bidirectional event-based communication

## Downloading
```bash
$ git clone https://github.com/kiahamedi/node-chat-app.git
$ cd node-chat-app/
$ npm install
$ npm start 
```
